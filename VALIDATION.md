# Validation Log

FactorForge predictions are in-silico only. This file tracks community wet-lab validation results.

**Try it first**: [factorforge-cds.vercel.app](https://factorforge-cds.vercel.app) — generate an optimized sequence in your browser before running it in the lab.

---

## How to Contribute

If you tested a FactorForge-optimized sequence in the lab, please share your results:

- Open a [GitHub Issue](https://github.com/eijex/factorforge-cds/issues) with the label `wet-lab-result`
- Or email: eijex.lab@gmail.com

Include:
- Host organism
- Input protein
- Optimized sequence (or FactorForge run parameters)
- Expression system / assay used
- Result (qualitative or quantitative)

Contributors will be credited below.
