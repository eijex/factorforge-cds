# v2 Rule Engine Archive

This directory preserves the historical deterministic rule engine that preceded
the current public `profile` engine namespace.

The archived code is not installed, registered, or supported as a runtime engine
in FactorForge v3.1.x. The supported production path is:

```text
src/factorforge/engines/profile/
```

The v2 archive is kept to document implementation history and to make the
profile-engine rename auditable. Small baseline artifacts are retained when they
help explain the former rule-engine behavior.
