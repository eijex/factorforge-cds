# FactorForge Agent Permission Model

## Agent may

- Read public docs, schemas, and synthetic examples
- Run local deterministic scripts
- Write review reports and task drafts

## Agent must never

- Self-approve or directly set an approval state
- Push to a public repository autonomously
- Expose raw biological sequences in output
- Generate yield or expression-success claims

## Human confirmation required

- Any persistent artifact write
- Public output generation
- Manuscript or README claim changes
